NLBNP API
=========

Use ``asunder.nlbnp.CorePeripheryPartition`` when core removal exposes final
connected-component communities. Use ``asunder.nlbnp.NonlinearBranchAndPrice``
when those components require finer subdivision. Use ``run_evaluation`` only
for the packaged case-study benchmark flow.
